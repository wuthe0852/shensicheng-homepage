乳腺癌诊断数据分析与预测模型构建

基于 Wisconsin Breast Cancer Dataset 的医学数据分析课程项目

项目背景

乳腺病灶良恶性判断是临床常见问题。为了展示标准的数据分析流程，本项目采用公开医学数据集，完成数据读取、预处理、描述性统计、模型构建、模型评估以及结果可视化。

分析流程

  • 读取公开医学数据集
  • 完成数据清洗、标准化和训练测试集划分
  • 进行描述性统计和相关性分析
  • 构建逻辑回归与随机森林模型
  • 使用 ROC、混淆矩阵和特征重要性进行评价

模型结果

Logistic Regression Accuracy: 0.9825
Logistic Regression AUC: 0.9954
Random Forest Accuracy: 0.9474
Random Forest AUC: 0.9937
